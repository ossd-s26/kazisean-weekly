This week in class we have learned about open source companies. Often open source can be easily mistaken for open source software, however open source have different types. The open source philosophy of free and open to anyone and everyone is adopted to some global projects. One of such is the [Internet Archive](https://archive.org/) project. This project is used to archive the internet. From books, pictures to even websites. The website is open for anyone and everyone to use without any charge. People around the world use this site to look at internet documents. The site is most popular for their way back machine which can be used to check the previous changes on a given website as internet archieve takes a snapshot of any website it can find for archival purposes.
